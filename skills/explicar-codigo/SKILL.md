name: explicar-codigo
description: Explicar um trecho de código de forma simples para quem está aprendendo
argument-hint: [caminho do arquivo ou trecho de código]
---

Explique o código em $ARGUMENTS para alguém que está aprendendo a programar.

Regras:
1. Se for um caminho de arquivo, leia o arquivo primeiro
2. Comece dizendo **o que o código faz** em uma frase simples
3. Depois explique **como ele faz**, percorrendo as partes principais
4. Use analogias do dia a dia quando possível
5. Destaque partes que podem confundir um iniciante
6. Evite assumir conhecimento prévio — explique termos técnicos na primeira vez que aparecem
7. Se o código tiver algum problema ou puder ser melhorado, mencione no final como "Dica"
